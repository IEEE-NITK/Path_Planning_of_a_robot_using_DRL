******FOR ADDING FILES*****
1. git add . 
2. git commit -m "commit message"
3. git push


******FOR CLONING REPO*****
1. make repo
2. copy clone link
3. goto location in git bash
4. git clone <shift+ins>
5. cd <cloned-dir>
6. copy file to cloned dir
7. git add . 
8. git commit -m "commit message"
9. git push
